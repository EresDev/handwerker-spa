# Handwerker SPA
[![<EresDev>](https://circleci.com/gh/EresDev/handwerker-spa.svg?style=svg)](https://app.circleci.com/pipelines/github/EresDev/handwerker-spa)

This is a front-end for Handwerker web application based on React. 

## Install the project

### `yarn install`

## Run tests

### `yarn test`

## Start locally

### `yarn start`
